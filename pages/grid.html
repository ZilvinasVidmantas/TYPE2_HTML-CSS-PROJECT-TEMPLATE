Grid

Grid, tai toks CSS savybės display tipas, kuris gali valdyti pirmo lygio vaikų išdėstymą tiek vertikaliai, tiek horiontaliai.
Taikydami papildomas savybes tėviniam elementui (tam, kuriam pritaikyta CSS savybė display: grid;) galime kurti stulpelius, eilutes ir netgi plotų šablonus(kuriuos nagrinėsime vėliau.)
Nustačius eilučių, stulpelių ar plotų šablonus, elementai yra išdėstomi pagal šiuos šablonus. Taip pat galima pirmo lygio vaikams nurodyti kokią "tinklelio" dalį/plotą jis užims.

grid-template-columns

Ši savybė nusako į kiek, ir kokio pločio, stulpelių bus suskirstomas "grid" elementas. 

        ... {
            display: grid;
            grid-template-columns: 100px 200px 100px 50px;
            pirmojo stulpelio plotis ↗     ↗    ↗     ↗
                antrojo stulpelio plotis ↗    ↗     ↗
                  trečiojo stulpelio plotis ↗     ↗
                    ketvirtojo stulpelio plotis ↗
        }

grid-template-rows

Ši savybė nusako į kiek, ir kokio pločio, eilučių bus suskirstomas "grid" elementas. 

        ... {
            display: grid;
            grid-template-rows: 100px 50px 100px;
       pirmosios eilutės aukštis ↗     ↗    ↗
           antrosios eilutės aukštis ↗    ↗
             trečiosios eilutės aukštis ↗ 
        }

grid-column

Ši CSS savybė nusako kiek stulplelių vaikinis elementas užims.
Tai galima nurodyti rašant span x, kur x - stulpelų skaičius.
Taip pat apibrėžti vaikinį elementą galima nurodant pradžios ir pabaigos stulpelių linijas.
Linijos numeruojamos iš kairės į dešinę, pradedant vienetu: 1, 2, ... ,n;
Taip pat linijos numeruojamos iš dešinės į kairę, tačiau su minuso ženklu: -1, -2, ...,-n; Linijų kiekis (n) - visada vienetu didesnis nei stulpelių skaičius. 

        .father {
          display: grid;
          grid-template-columns: 100px 100px 100px 100px;
        }
    
        /* vaikinis elementas užims 2 stulpelius */
        .child-1 {
          grid-column: span 2; 
        }
        
        /* elementas prasidės nuo 2 linijos ir pasibaigs ties ketvirta */
        .child-2 {
          grid-column: 2 / 4;
        }
        
        /* elementas prasidės nuo 2 linijos ir pasibaigs paskutine */
        .child-3 {
          grid-column: 2 / -1;
        }

grid-row

Ši savybė aprašo kiek ir kokių eilučių užims "grid" vaikinis elementas. Principai veikia taip pat kaip ir savybėje grid-column

Grid plotai

Sudarius tinklelį naudojant stulpelius ir eilutes yra automatiškai sukuriamos linijos.
grid-area tai keturiomis linijomis apibrėžtas vaikinio elemento plotas - viršutinė, kairė, apatinė ir dešinė linijos. Jis visuomet turi būti stačiakampio formos.
Grid pirmo lygio vaikų plotą taip pat galime apibrėžti žodiniu pavidalu, kuomet tėviniame elemente esame aprašę plotų šabloną.

grid-area

Ši savybė taikoma vaikiniam "grid" elementui. Plotas gali būt aprašomas linijomis, arba pagal ploto pavadinimą (ploto pavadinimas turi būti aprašytas tėvinio "grid" šablone.). 

                            .child-1 {
                              grid-area: 2 / 4 / 4 / 6;
              viršutinė eilutės linija  ↗   ↗  ↗  ↗
                    kairė stupelio linija ↗  ↗  ↗
                    apatinė eilutės linija ↗  ↗
                    dešinė stulpelio linija ↗
                            }
    
                            Vaikinis plotas įvardintas pagal tėvinio Grid plotų šabloną
                            .child-2 {
                              grid-area: navbar;
                            }
    
                            Vaikinis plotas įvardintas pagal tėvinio Grid plotų šabloną
                            .child-3 {
                              grid-area: sidebar;
                            }

grid-template-areas

Ši savybė taikomai "grid" tėviniam elementui. Ši savybė sudaroma aprašant plotų šablonus mums patogiais raktažodžiais.
Sudarant plotų šabloną, svarbu įvardinti plotus taip, jog jie būtų stačiakampiai, priešingu atveju, šablonas neveiks.
Kiekvienos eilutės šablonas turi būti tarp dvigubų kabučių, pvz.: "raktažodis raktažodis1". Jungiant eilutes galime sudaryti visą 2D plotų šabloną. Raktažodžiai, gali kartotis skirtingose eilutėse.
Stulpelių kiekis, kiekvienoje eilutėje privalo būti vienodas! 

              .grid-mobile {
                display: grid;
                grid-template-columns: 1fr; // Aprašome vieną stulpelį
                grid-template-rows: 60px 60px auto auto auto 60px;// 6 eilutės
                grid-template-areas:
                  "header" // pirmos eilutės šablonas
                  "nav" // antros eilutės šablonas
                  "main" // trečios eilutės šablonas
                  "ads" // ketvirtos eilutės šablonas
                  "gallery" // penktos eilutės šablonas
                  "footer"; // šeštos eilutės šablonas
              }
    
              .grid-tablet {
                display: grid;
                grid-template-columns: 200px auto; // Aprašome 2 stuleplius
                grid-template-rows: 60px 60px auto auto 60px;// 5 eilutės
                grid-template-areas:
                  "header header" // pirmos eilutės šablonas
                  "nav    nav" // antros eilutės šablonas
                  "main   main" // trečios eilutės šablonas
                  "ads    gallery" // ketvirtos eilutės šablonas
                  "footer footer"; // penktos eilutės šablonas
              }
    
              .grid-desktop {
                display: grid;
                grid-template-columns: 250px 600px 400px auto ; // Aprašome 4 stuleplius
                grid-template-rows: 60px auto auto 60px;// 4 eilutės
                grid-template-areas:
                  "header header  header  nav" // pirmos eilutės šablonas
                  "ads    main    main    nav" // antros eilutės šablonas
                  "ads    gallery gallery nav" // trečios eilutės šablonas
                  "footer footer  footer  footer" // ketvirtos eilutės šablonas
              }

grid-template

Tai savybių grid-template-columns, grid-template-rows ir grid-template-areas apjungimas. 

              .grid-desktop {
                display: grid;
                grid-template:
                  "header header  header  nav" 60px
                  "ads    main    main    nav" auto
                  "ads    gallery gallery nav" auto
                  "footer footer  footer  footer" 60px /
                   250px  600px   400px   auto
              }

Įvardintos linijos

Kartais sudarant tinklelio plotų šablonus (angl.: grid-template-areas) norima sudaryti persiklojančius plotus. Tokiais atvejais, nėra įgalintos galimybės sukurti ploto šablono, kuriame tą pačią vietą užimtų skirtingi plotai/sekcijos. Šiai problemai spręsti yra naudojama įvardintų tinklelio linijų metodologija.

Bet kuriai tinklelio linijai galima suteikti papildomus pavadinimus. Tuomet vaikiniam elementam yra aprašomas plotas pagal linijų apibrėžtumą: iš viršaus, kairės, apačios ir dešinės. Įvardinant plotus tokiu būdu, galime įgalinti sekcijų persiklojimą.

Apibrėžus tinklelio vaikinius elementus įvardintomis linijomis, galime keisti jų išdėstymus aprašant eilučių ir stulpelių šablonus tėviniam elementus ir pervadinant kitas linijas, tokiais pat pavadinimais, kokiomis yra apibrėžti vaikiniai elementai.
